## ‚úÖ Solution

### Issue 1: Remove Scrollbar

The scrollbar appears because there's horizontal overflow. Fix this:

```css
/* Add to your main container or global CSS */
.overflow-x-hidden {
  overflow-x: hidden;
}

/* Or explicitly hide scrollbar */
.hide-scrollbar {
  -ms-overflow-style: none;  /* IE and Edge */
  scrollbar-width: none;  /* Firefox */
}

.hide-scrollbar::-webkit-scrollbar {
  display: none;  /* Chrome, Safari, Opera */
}
```

Apply to your main container:

```tsx
<div className="flex flex-col h-screen overflow-x-hidden">
  {/* Your content */}
</div>
```

---

### Issue 2: Make Content Full Width

Your lyric cards and content boxes should use `w-full`:

**Before:**
```tsx
<div className="p-4 rounded-lg bg-card">  {/* Not full width */}
  Que sepas lo mucho que te quiero
</div>
```

**After:**
```tsx
<div className="w-full p-4 rounded-lg bg-card">  {/* Full width */}
  Que sepas lo mucho que te quiero
</div>
```

Also ensure parent containers don't have padding that creates the scrollbar:

```tsx
{/* Lyrics container */}
<div className="w-full px-0">  {/* Remove horizontal padding */}
  {/* Lyric cards */}
</div>
```

## üîß CSS Fixes

Add these to your global CSS or Tailwind config:

```css
/* Hide scrollbar globally if needed */
* {
  scrollbar-width: none; /* Firefox */
  -ms-overflow-style: none; /* IE/Edge */
}

*::-webkit-scrollbar {
  display: none; /* Chrome/Safari */
}

/* Or target specific containers */
.lyrics-container {
  overflow-x: hidden;
  width: 100%;
}

/* Prevent horizontal scroll */
body {
  overflow-x: hidden;
  max-width: 100vw;
}

/* Ensure full width on mobile */
@media (max-width: 768px) {
  .container {
    max-width: 100% !important;
    padding-left: 0 !important;
    padding-right: 0 !important;
  }
}
```
